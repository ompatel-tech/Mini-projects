#  Python Countdown Timer

##  Description

A simple Python mini project that allows users to enter time in seconds and displays a live countdown in **HH:MM:SS format** until the timer reaches zero.

##  Features

* Takes user input for countdown time
* Converts seconds into hours, minutes, and seconds
* Updates the timer every second
* Displays **"TIMES UP"** when finished

##  Tech Stack

* Python
* Built-in `time` module
